# Salary-Data-Analysis


## 📌 Project Overview
This project analyzes salary data of 22,000+ job records to identify key factors influencing salary such as job role, location, company rating, and employment type.

## 🎯 Objectives
- Analyze salary trends across different roles and cities
- Understand correlation between rating and salary
- Identify high-paying job categories
- Build a salary prediction model

## 📊 Dataset
- Rows: ~22,770
- Columns: 8
- Features:
  - Rating
  - Company Name
  - Job Title
  - Salary
  - Salaries Reported
  - Location
  - Employment Status
  - Job Roles

## 🔍 Key Insights
- Senior roles earn significantly higher salaries
- Location is a strong salary predictor
- Weak correlation between rating and salary
- Full-time jobs pay more than contract roles

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
